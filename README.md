### Prerequisites

* Node.js 16 or newer
* npm

### Install

```bash
# clone the repo
git clone https://github.com/mohdkamran-khan/Airbnb-Frontend.git
cd Airbnb-Frontend

# install dependencies
npm install
```

### Run in development

```bash
npm run dev
# open http://localhost:5173 or the port Vite prints
```

### Build for production

```bash
npm run build
# preview production build locally
npm run preview
```
